Overview

The strut.less project is an attempt to bring everything we love about modern responsive design under one roof. The goal is to provide simple structures for prototyping and designing great content, with as little overhead as possible. Prefering not to predefine elements for you, strut gives you classes and mix-ins that are re-usable throughout any of your web-design projects.

This framework uses normalize.css to give us a baseline across all browsers.

Grid System

The grid system is based off a 12 column layout and will resize to fit the container it is in. Each column div must be placed in a row. It is optional to place it into a conatainer but the gutter padding on the outside won"t display unless you do.

Each grid layout can be made to be responsive by using additional classes like the ones here grid collapse. Columns that are not set to collapse will have their overflow hidden by default

Colors

Like elsewhere in Strut.less colors in this section can be applied using to the text and the background using the suffix "text-" and "bg-".

ROYGBIV

The default colors in this framework are built around the widely used colors at flatuicolors.com. The color names have been standardized so they are easy to use and remember as class names and LESS variables.

Shades of Grey

These greys give designers a wide range of greys in a simple to use format, similar to the ROYGBIV colors. Each grey is 12.5% darker than the last and scale from white to black.
